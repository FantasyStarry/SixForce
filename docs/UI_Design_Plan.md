# SixForce UI 重设计方案

## 一、设计理念

### 核心风格定位
- **工业科技感**：专业、精准、可靠的工业软件气质
- **简约现代**：去除冗余装饰，突出功能本身
- **高效操作**：信息层级清晰，操作路径最短
- **视觉舒适**：长时间使用不疲劳，数据易读

### 设计原则
1. **功能优先**：UI服务于功能，不为设计而设计
2. **信息密度适中**：专业软件需要展示足够信息，但不拥挤
3. **状态清晰**：设备连接、数据采集等状态一目了然
4. **操作反馈**：每个操作都有明确的视觉反馈

## 三、布局重构

### 整体布局：三栏式 + 顶部导航

```
┌─────────────────────────────────────────────────────────────┐
│  顶部导航栏 (56px)                                           │
│  [Logo] [产品选择] [状态指示器] [主题切换] [设置]              │
├─────────┬───────────────────────────────────┬───────────────┤
│ 左侧栏   │         中央主区域                 │   右侧栏      │
│ (220px) │         (自适应)                   │   (300px)     │
│         │                                   │               │
│ 连接配置 │    ┌─────────────────────────┐    │  实时数据     │
│         │    │     数据可视化区域       │    │  表格显示     │
│ 串口    │    │     (曲线图/仪表盘)      │    │               │
│ 波特率  │    │                         │    │  参数设置     │
│ 机码    │    └─────────────────────────┘    │               │
│ 产品编号│                                   │  高级功能     │
│ 通道    │    ┌─────────────────────────┐    │               │
│         │    │       图例/控制区        │    │               │
│ 连接状态│    └─────────────────────────┘    │               │
│         │                                   │               │
│ [连接]  │                                   │               │
│ [断开]  │                                   │               │
└─────────┴───────────────────────────────────┴───────────────┘
```

### 布局改进点

1. **顶部导航栏**
   - 高度从当前的约60px调整为56px，更紧凑
   - Logo区域增加品牌辨识度
   - 状态指示器更醒目，使用动态效果
   - 增加快捷设置入口

2. **左侧面板**
   - 宽度从200px调整为220px，表单更舒适
   - 分组更清晰：连接配置 / 设备参数 / 操作按钮
   - 连接状态使用更大的视觉指示器

3. **中央区域**
   - 曲线图区域增加工具栏（缩放、导出、暂停等）
   - 图例区域可折叠
   - 增加数据统计摘要（最大值、最小值、平均值）

4. **右侧面板**
   - 数据表格优化，数值更醒目
   - 参数设置区域更紧凑
   - 高级功能区域增加更多入口

## 四、组件设计

### 4.1 顶部导航栏

```
设计要点：
- 毛玻璃效果背景（深色主题下）
- Logo使用渐变色图标
- 产品选择下拉框带搜索功能
- 状态指示器：脉冲动画表示在线
- 主题切换：平滑过渡动画
```

### 4.2 连接配置面板

```
设计要点：
- 表单项使用图标前缀
- 下拉框支持键盘导航
- 连接状态使用大型指示器（圆形，带呼吸灯效果）
- 按钮组：主按钮（连接）+ 次按钮（断开）
- 连接成功后显示连接时长
```

### 4.3 数据可视化区域

```
曲线图设计：
- 深色背景网格（微弱的网格线）
- 曲线使用渐变色 + 发光效果
- 鼠标悬停显示数据点详情
- 支持区域缩放和平移
- 右上角工具栏：暂停/继续、导出、全屏

图例设计：
- 水平排列，可点击切换显示/隐藏
- 使用圆角标签样式
- 颜色块 + 通道名 + 当前值
```

### 4.4 实时数据表格

```
设计要点：
- 表头固定，内容滚动
- 数值使用等宽字体
- 正负值使用不同颜色
- 数值变化时有闪烁动画
- 支持点击复制数值
```

### 4.5 按钮系统

```
按钮类型：
1. 主按钮：青绿色填充，用于主要操作
2. 次按钮：边框样式，用于次要操作
3. 危险按钮：红色边框/填充，用于危险操作
4. 图标按钮：仅图标，用于工具栏

按钮状态：
- 默认：标准样式
- 悬停：轻微上浮 + 阴影
- 按下：下沉效果
- 禁用：50%透明度
- 加载中：旋转图标
```

## 五、交互设计

### 5.1 连接流程

```
1. 选择串口 → 自动检测可用串口，显示设备名称
2. 配置参数 → 波特率、机码等
3. 点击连接 → 按钮显示加载状态
4. 连接成功 → 状态指示器变绿，开始数据采集
5. 连接失败 → 显示错误提示，提供重试选项
```

### 5.2 数据采集流程

```
1. 连接成功后自动开始采集
2. 曲线实时滚动更新
3. 表格数据同步更新
4. 支持暂停/继续采集
5. 支持数据记录和导出
```

### 5.3 反馈机制

```
- Toast通知：操作成功/失败提示
- 加载状态：按钮/区域加载指示
- 进度条：长时间操作进度显示
- 确认对话框：危险操作二次确认
```

## 六、动效设计

### 6.1 过渡动画

```
- 页面切换：淡入淡出 (200ms)
- 面板展开/收起：滑动 (250ms)
- 弹窗出现：缩放 + 淡入 (200ms)
- 主题切换：颜色渐变 (300ms)
```

### 6.2 状态动画

```
- 连接状态：呼吸灯效果 (2s循环)
- 数据更新：数值闪烁 (100ms)
- 按钮悬停：上浮 (150ms)
- 加载中：旋转图标 (1s循环)
```

### 6.3 曲线动画

```
- 新数据点：平滑过渡
- 曲线滚动：流畅滚动
- 缩放：平滑缩放
```

## 九、视觉效果预览

### 深色主题效果描述

整体呈现深邃的太空蓝色调，主色采用青绿色（#00D9C0），给人专业、科技、可靠的感觉。

- 顶部导航栏带有微妙的毛玻璃效果
- 卡片使用轻微的边框和阴影，层次分明
- 曲线图区域有微弱的网格背景，曲线带有发光效果
- 数据表格数值醒目，使用等宽字体
- 按钮有精致的悬停和点击效果
- 状态指示器使用动态呼吸灯效果

### 与当前设计的主要区别

| 方面 | 当前设计 | 新设计 |
|------|----------|--------|
| 主色 | 蓝紫色 #6B8AFF | 青绿色 #00D9C0 |
| 背景 | 深灰 #0F1419 | 深空蓝 #0A0E14 |
| 风格 | 标准Material | 工业科技感 |
| 动效 | 基础 | 丰富但克制 |
| 状态指示 | 静态 | 动态呼吸灯 |
